# QR Code Generator (Terminal)

A simple Python terminal-based QR code generator using the open-source library by [Project Nayuki](https://www.nayuki.io/page/qr-code-generator-library).  

This project allows you to input text or binary data and generate QR codes that can be saved as image files.

---

## Features

- Generate QR codes from text or data input via the terminal.
- Save QR codes to an image file in a location of your choice.
- Supports letters, numbers, hyphens, and underscores.
- Lightweight and easy to run with Python 3.
